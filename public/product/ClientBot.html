Documentation & Warnings

Architecture

This client operates in LocalStorage mode. Messages sync across browser tabs instantly via the Storage Event API. No external network requests are made.

Security Notice
  • Do not paste production tokens in shared environments.
  • Local storage is accessible by any script on this domain.
  • Auto-cleanup runs every 24h (simulated).
Rate Limits

Client enforces a 3-second cooldown between dispatches to simulate API fairness. Exceeding triggers a soft lock.
