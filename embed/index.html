Embed Tools

Встраивайте инструменты лаборатории на свой сайт через iframe. Без API, без регистрации, полностью локально.

Все инструменты работают исключительно в браузере пользователя. При встраивании данные не передаются на наши серверы. Файлы, текст и настройки остаются локальными.

Universal Template

Базовый шаблон для встраивания любой страницы лаборатории.

<iframe
src="https://seraviel-labs.vercel.app/{PATH}"
width="100%"
height="600"
frameborder="0"
loading="lazy"
allow="clipboard-write"
></iframe>

Рекомендации

  • Используйте loading="lazy" для отложенной загрузки и ускорения страницы
  • Добавляйте атрибут title для доступности: title="SERAVIEL LABS Tool"
  • Для адаптивности оборачивайте iframe в контейнер с position: relative; padding-bottom: 75%
  • Атрибут allow="clipboard-write" необходим только для инструментов с копированием
  • Изменение содержимого iframe через JS невозможно из-за политики безопасности браузера (CORS)
