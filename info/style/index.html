About SERAVIEL LABS

SERAVIEL LABS is a collection of client-side developer tools built with a consistent design philosophy:

  • 0 API — all tools work offline, no data leaves your browser
  • 0 emojis — only clean, scalable SVG icons
  • toast() only — consistent, non-blocking notifications
  • Single file — each tool is one self-contained index.html

👉 Want to see the design system? Add ?style to the URL.

How to Use

  1. Copy the :root block from this page's <style> into your project.
  2. Use variables like color: var(--text) instead of hard-coded values.
  3. For icons: copy the <path> from any icon card and wrap in <svg viewBox="0 0 24 24" ...>.
  4. Keep the toast() function for consistent notifications across all SERAVIEL LABS tools.
