Команды:
install <url> [опции] Установить пакет
--sha256 <hash> Проверить целостность
--add-to-path Добавить в PATH
--dry-run Показать действия без выполнения
remove <name> Удалить пакет
list Показать установленные пакеты
search <query> Поиск по установленным
config set/get Управление настройками
clear / cls Очистить экран
help Показать эту справку
version Версия installer-s
exit / quit Выйти
 
Глобальные флаги:
--silent, -s Тихий режим
--log=<level> debug|info|warn|error
--uninstall Удалить installer-s
 
> _
>
Внимание: Installer-S предназначен только для Windows. Для других ОС используйте альтернативные методы установки.

SHA-256 Verification

Автоматическая проверка целостности каждого пакета перед установкой.

Native Performance

Скомпилирован в .NET 8. Работает мгновенно, без зависимостей и рантаймов.

Clean Uninstall

Полное удаление через встроенный деинсталлятор. Никаких следов в системе.

Silent Mode

Поддержка автоматизации через флаги --silent и --log для CI/CD пайплайнов.

Скачай установщик

Запусти от админа

3

Используй CLI

Открой терминал и введи installer-s. Готово к работе.
